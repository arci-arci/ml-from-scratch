A simple repository created with the idea of learning Go and Machine Learning by creating from scrath different type of ML algorithms.
This is a (non comprehensive) list of implemented ML algorithms:

Dataset used for classification: [Enron Spam](https://www2.aueb.gr/users/ion/data/enron-spam/)

- Naive Bayes classifier
- KNN
  - Next possible implementation: [Ball Tree](https://web.archive.org/web/20251219030314/https://citeseerx.ist.psu.edu/document?repid=rep1&type=pdf&doi=17ac002939f8e950ffb32ec4dc8e86bdd8cb5ff1), [Wikipedia](https://en.wikipedia.org/wiki/Ball_tree)
- K-Means
- Decision tree
- Linear regression and classification
